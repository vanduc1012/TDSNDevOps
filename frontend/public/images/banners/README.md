# Thư mục Banner

Thư mục này chứa các ảnh banner quảng cáo cho trang chủ.

## Cách sử dụng:

1. **Thêm ảnh banner:**
   - Đặt ảnh banner vào thư mục này
   - Đặt tên file là `banner.jpg` (hoặc `banner.png`)
   - Kích thước khuyến nghị: 1200x400px hoặc tỷ lệ tương tự

2. **Định dạng ảnh hỗ trợ:**
   - JPG/JPEG
   - PNG
   - WebP

3. **Nhiều banner:**
   - Nếu có nhiều banner, đặt tên: `banner1.jpg`, `banner2.jpg`, ...
   - Sau đó cập nhật code trong `Home.js` để hiển thị carousel

## Ví dụ tên file:
- `banner.jpg` - Banner chính
- `banner1.jpg` - Banner 1
- `banner2.jpg` - Banner 2

## Lưu ý:
- Ảnh sẽ tự động resize để phù hợp với container
- Nếu không có ảnh, sẽ hiển thị text fallback
- Kích thước file nên < 500KB để tối ưu tốc độ tải trang
